# Web dev bootcamp

Various small projects built during the web development course of Colt Steele: https://www.udemy.com/course/the-web-developer-bootcamp/

Topics covered

- HTML5
- CSS3
- JavaScript
- Bootstrap
- DOM Manipulation
- jQuery
- Unix(Command Line) Commands
- NodeJS
- NPM
- ExpressJS
- REST
- MongoDB
- Database Associations
- Authentication
- PassportJS
- Authorization
